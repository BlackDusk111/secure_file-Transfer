# secure_file-Transfer
SecureTransfer is a Python desktop application for secure peer-to-peer file transfer, developed as coursework for the ST5062CEM Programming and Algorithm 2 module at Softwarica College of IT and E-Commerce. It uses hybrid RSA-2048 and AES-256-CBC encryption, generating a fresh key and IV for every file so the server can route packets without reading them, and applies SHA-256 hashing so recipients can verify file integrity. User accounts are stored locally with SHA-256 hashed passwords, and the attack detection module handles three threats at once: a rate limiter blocking IPs exceeding ten requests in ten seconds for DoS protection, a nonce-and-timestamp check for replay attacks, and a public key fingerprint alert system for MITM detection. The multi-threaded TCP server supports multiple users, each on a separate thread, while the Tkinter GUI provides a file hub with an activity log, transfer history, and a live Attack Monitor showing event counts and logs in real time. To run, install the dependency with pip install cryptography==42.0.5, start the server with python server.py, open client windows with python client.py, register a username, and run all tests with python -m unittest test_securetransfer -v.
